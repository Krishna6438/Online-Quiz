# 🎯 Online Quiz Competition Web App

A modern, interactive, and responsive **Online Quiz Competition Web Application** built using **HTML, CSS, and JavaScript**. This project simulates a real-world quiz platform with a sleek UI, scoring system, timer, and leaderboard — all running purely on the frontend.

---

## 🚀 Features

### 🏠 Landing Page
- Attractive hero section with call-to-action
- Smooth animations and responsive design
- "Start Quiz" button to begin

### 👤 User System (Frontend-Based)
- Enter username before starting quiz
- Stores user data using `localStorage`

### 🧠 Quiz System
- Multiple Choice Questions (MCQs)
- Questions stored in JSON format
- One question displayed at a time
- Timer for each question
- Auto-transition to next question
- Visual feedback on selected answers

### 📊 Scoring System
- Score calculated based on correct answers
- Optional bonus points for faster responses
- Final score displayed at the end

### 📈 Progress Tracking
- Progress bar (Question number / Total)
- Question counter display

### 🏆 Result Page
- Final score summary
- Correct vs incorrect answers
- Performance feedback (Excellent / Good / Try Again)
- Restart quiz option

### 🥇 Leaderboard
- Stores top scores using `localStorage`
- Displays top 5 players
- Automatically sorted by highest score

### 🔊 Sound Effects (Optional)
- Correct answer sound
- Wrong answer sound

### 🎨 UI/UX Design
- Modern and clean interface
- Gradient / glassmorphism styling
- Smooth transitions and animations
- Fully responsive (mobile + desktop)

### 🌙 Advanced Features (Optional)
- Dark mode toggle
- Category selection
- Difficulty levels
- Countdown animation
- Confetti effect for high scores

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Animations
- **JavaScript (Vanilla)** – Logic & Interactivity
- **LocalStorage** – Data persistence (no backend)
